Installation

# Homebrew (macOS / Linux)
$ brew install nullclaw/tap/nullclaw

# Build from source
$ git clone https://github.com/nullclaw/nullclaw
$ cd nullclaw
$ zig build -Doptimize=ReleaseSafe

# Verify
$ nullclaw --version
nullclaw 1.0.0 (678 KB)

Setup

# Interactive onboarding — sets up provider, channel, memory
$ nullclaw onboard --interactive

? Choose your AI provider: Anthropic
? Enter API key: sk-ant-***
? Choose channel: Telegram
? Enable memory? Yes

✓ Configuration saved to ~/.config/nullclaw/config.json
✓ Ready to go!

Daemon Mode

Always-on background operation

# Start the daemon
$ nullclaw daemon --start
✓ Daemon running (PID 4821, 0.9 MB RAM)

# Check status
$ nullclaw daemon --status
Uptime: 14d 3h | Messages: 12,847 | Memory: 0.9 MB

# Graceful stop
$ nullclaw daemon --stop
  • Persists across reboots via systemd / launchd
  • Sub-1 MB memory footprint — runs on anything
